vscode ME
CTRL+C, CTRL+V = COPY, PASTE

CTRL+A = SELECT ALL (jabh puri file ka jitna text hai sabko copy karna ho)

CTRL+D = REPLICATE SELECTED PORTION(line copy karke nichhe chala jaega)

CTRL+BACKSPACE = PURA PURA WORD EK SATH DLT HOGA

SHIFT+ENTER = ADD LINE BELOW (ek line niche bana deta jabh kabhi kisi code ke beech me ho or waha se nae line leni ho)

CTRL+SHIFT+ENTER = ADD LINE ABOVE(phele jaisa hai bas uper line enter kar deta)

ALT + CURSOR_CLICK = MULTIPLE CURSOR(ek hi time pe bhhot jagaha cursor. kahi se cursor hatana ho to alt dabake us jagaha wale cursor pe click kardo)

CTRL+Z = UNDO

CTRL+Y = REDO

F2 = TO RENAME A FILE (kisi file me click karo or f2 dabao rename karne ke liye)

ALT DOWNARROW/UPPERARROW = LINE UPER NICHE CHALI JAEGI
VS CODE ME left side me sabse niche dekho settings milegi. usko click karke usme ek option hai keyboard shortcut. usme click karke jo shortcut apko chahiye usko search mar lo (bas vs code ke liye)

YOUTUBE shortcut:

SPACEBAR = TO PLAY/PAUSE VIDEO

SHIFT + <(GREATER THAN) = TO DECREASE SPEED OF VIDEO

J = 10 SEC PICHHE

UPPER ARROW = TO INCREASE VOLUME

LOWER ARROW = TO DECREASE VOLUME
